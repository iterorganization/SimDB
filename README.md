# ITER Simulation Management Tool

## CLI

### Dependencies

* Python >= 3.6
* PyYAML >= 3.12
* SqlAlchemy >= 1.2.2

### Installing the CLI

```bash
pip install --user git+ssh://git@git.iter.org/imex/simulation-management.git
```

## Remote API

* Python >= 3.6
* SqlAlchemy >= 1.2.2
* Flask >= 0.12.2
* PostgreSQL >= 9.6
* psycopg2-binary >= 2.7.4
